# Keyboard Layout & Finger Mapping

## Standard QWERTY Finger Assignment

### Left Hand
| Finger | Keys |
|--------|------|
| Pinky | `1`, `Q`, `A`, `Z` |
| Ring | `2`, `W`, `S`, `X` |
| Middle | `3`, `E`, `D`, `C` |
| Index | `4`, `R`, `F`, `V`, `5`, `T`, `G`, `B` |

### Right Hand
| Finger | Keys |
|--------|------|
| Index | `6`, `Y`, `H`, `N`, `7`, `U`, `J`, `M` |
| Middle | `8`, `I`, `K`, `,` |
| Ring | `9`, `O`, `L`, `.` |
| Pinky | `0`, `P`, `;`, `/`, `-`, `=`, `[`, `]`, `\`, `'` |

### Thumbs
Both thumbs share the Space bar.

## Home Row Position
Left hand: **A S D F** (index on F)
Right hand: **J K L ;** (index on J)

F and J have physical bumps on most keyboards — these are the "home" landmarks.

## Lesson Progression

### Beginner (Lessons 1-8)
1. Home row left: F D S A
2. Home row right: J K L ;
3. Home row combined: all 8 keys
4. Add G and H (index finger reaches)
5. Add R and U (index finger reaches up)
6. Add T and Y (index finger reaches up)
7. Home row + reaches review
8. Common words with home row keys

### Intermediate (Lessons 9-16)
9. Top row left: Q W E R
10. Top row right: I O P
11. Top row combined
12. Bottom row left: Z X C V
13. Bottom row right: M , . /
14. Bottom row combined
15. Numbers row
16. Common bigrams and trigrams

### Advanced (Lessons 17-24)
17. Shift keys (capital letters)
18. Punctuation basics (. , ! ?)
19. Brackets and symbols
20. Numbers fluency
21. Common English words (top 100)
22. Common English words (top 200)
23. Code snippets (for programmers)
24. Speed drills
